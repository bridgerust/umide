Features

Everything you need, in one place

📱

Embedded Emulators

Android Emulator and iOS Simulator live inside the IDE as panels. Launch, stop, and interact without ever leaving your editor.

⚡

Built in Rust

Native performance with a GPU-accelerated UI built on Floem. Starts instantly, stays fast regardless of project size.

🔀

Cross-Platform Projects

First-class support for React Native and Flutter. Work on both platforms from a single unified environment.

🧠

LSP Support

Full Language Server Protocol support for code completion, diagnostics, go-to-definition, and code actions out of the box.

🖥️

Built-in Terminal

Run metro, flutter, adb commands and more without leaving UMIDE. The terminal is always one click away.

🔌

Plugin System

Extend UMIDE with plugins written in any language that compiles to WASI — Rust, C, AssemblyScript and more.

Built with

Rust C++ Objective-C++ Floem WGPU LSP WASI

Built on Lapce

A fully-fledged IDE, not a viewer

UMIDE is built on top of Lapce, a production-grade Rust editor. You get everything Lapce offers — LSP, vim mode, remote development, plugin system, split editors, and a built-in terminal — with Android and iOS emulators embedded directly alongside your code.
